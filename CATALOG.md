# Content Catalog

This catalog provides a centralized, searchable inventory of all enablement content available in this repository.

It covers:
- Labs (hands-on, technical)
- Playbooks (prescriptive guides)
- Trainings (structured learning)
- Workshops (end-to-end sessions)

Use browser search (Ctrl/Cmd + F) to filter by:
- Technology: Snowflake, Databricks, GenAI, Azure, AWS, GCP  
- Type: Lab, Playbook, Training, Workshop  
- Level: Beginner, Intermediate, Advanced  
- Domain: Retail, BFSI, Pharma, Platform, etc.
